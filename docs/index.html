Clipboard Investigator

A macOS desktop app to inspect and analyze clipboard contents in detail. See every data type, preview images, and explore binary payloads.

Tauri 2.3 Rust macOS MIT License

Features

Read Clipboard

Click a button or press Cmd+V to read all clipboard types via the native NSPasteboard API.

Drag & Drop

Drop files into the window to inspect their DataTransfer details including MIME types and sizes.

Image Preview

Automatically detects image data (PNG, JPEG, TIFF, etc.) and shows inline thumbnails.

Binary Data

Non-text clipboard entries are base64-encoded and displayed with size information.

Type Identification

Displays the UTI (Uniform Type Identifier) for each clipboard entry on macOS.

Lightweight

Vanilla JS frontend with no framework dependencies. Fast startup, minimal resource usage.

How It Works

Clipboard Investigator is built with Tauri 2, combining a Rust backend for native clipboard access with a vanilla JavaScript frontend rendered in a system WebView.

Clipboard Entry Structure

Each clipboard type is returned as a ClipboardEntry:

{
  "type_name": "public.utf8-plain-text",  // UTI string
  "data": "Hello, world!",                // text or base64-encoded binary
  "is_text": true,                         // content type flag
  "size": 13                               // byte count
}

Text Detection

The backend classifies UTIs as text if they contain any of these keywords: text, string, url, html, xml, json, yaml, plist, rtf, css, javascript, source-code, or plain-text. Everything else is treated as binary and base64-encoded.

Getting Started

Prerequisites

  • Rust 1.77.2 or later — install via rustup
  • macOS (clipboard reading uses NSPasteboard; other platforms return empty data)
  • Tauri CLI — install with cargo install tauri-cli --version "^2"

Clone the Repository

git clone https://github.com/vijaykrishnavanshi/clipboard-investigator.git
cd clipboard-investigator

Run in Development Mode

cd src-tauri
cargo tauri dev

This launches the app with hot-reload for the frontend. The window opens at 1024×768.

Build for Production

cd src-tauri
cargo tauri build

The built app bundle is at:

src-tauri/target/release/bundle/macos/Clipboard Investigator.app
